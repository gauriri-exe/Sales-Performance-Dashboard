# Sales-Performance-Dashboard

Project Description:
Interactive Power BI dashboard to convert sales data into parsable and actionable business intelligence. The dashboard allows users to see a high-level overview of revenue performance, profitability and key market trends to allow for data driven decision making.

Tech Stack:
The dashboard will be used to conduct visualisation in Power BI Desktop, perform data transformation via Power Query (M Language), analytical tools will consist of DAX (data analysis expressions) and Star Schema. UI/UX design will be modern and adhere to CY25SU12 design guidelines for professional and data driven storytelling.

Data Source:
The data source includes integrated sales transaction, product catalogue and regional sales transaction data. All data will be pulled from SQL based query statements for structured data retrieval.


Features and Highlights:
> Dynamic KPIs - The dashboard allows users to track real-time total revenue, profit margins and sales volumes.
> Geospatial Analysis - The dashboard will have an interactive map style design that will allow users to identify regions with the highest growth potential.
>Trend Forecasting - The dashboard will allow users to identify seasonal trends and changes in year over year performance.
> Drill Down Capabilities - The dashboard will allow users to utilize advanced filter capabilities to analyse performance by product or specific time period.
